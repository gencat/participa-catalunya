# Decidim::ResourceBank

.

## Usage

ResourceBank will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-resource_bank"
```

And then execute:

```bash
bundle
```
